## 1. 登录与初始化

### 1.1 访问地址

浏览器打开：`http://<部署服务器 IP>:8000/`

例：`http://172.16.50.100:8000/`

支持 Chrome / Edge / Safari 现代版本。**不支持 IE**。

### 1.2 首次登录

| 默认账号 | 默认密码 |
|---|---|
| `admin` | `admin123` |

**⚠️ 登录后第一件事——立刻改密码**：

```
左侧导航「后台管理」→ 用户 → 找到 admin 行 → 点钥匙图标 → 改密码
```

### 1.3 角色说明

| 角色 | 权限 |
|---|---|
| `super_admin`（超级管理员） | 所有权限 + 解锁定稿底稿 |
| `auditor`（审查员） | 创建任务 / 复核 / 定稿 / 改密码 |
| `unit`（被检查单位） | 只能看自己单位的任务 / 上传材料 |
| `readonly`（只读用户） | 只看不改 |

## 2. 系统初始化（管理员一次性操作）

如果是**全新部署**的系统，按以下顺序灌入黄金数据：

### 2.1 评价指标库（54 项）

部署脚本会自动 seed 54 项指标。如果没有，进入：

```
左侧导航「评价指标」→ 右上「批量导入 (JSON / PDF / Word / Excel)」
→ 上传官方核查工作底稿 .xlsx 文件
→ 系统识别表头自动解析（绿色 banner「Excel 表头自动识别」）
→ 「确认导入」
```

### 2.2 问题清单库

```
左侧导航「问题清单」→ 右上「批量导入」
→ 上传 check_items_seed.json
```

### 2.3 法规库

```
左侧导航「法规库」→ 右上「上传法规」单文件 / 「文件夹批量上传」
→ AI 自动识别文档类型（上位法 / 评价办法 / 编报指南 / 地方法规 等）
→ 自动入向量库供 RAG 检索
```

支持格式：PDF / Word / TXT / Markdown

### 2.4 配置 AI 大模型

```
左侧导航「后台管理」→ 大语言模型
→ 提供商选「DeepSeek」
→ 填 API Key（在 https://platform.deepseek.com/ 申请）
→ 模型选 deepseek-chat
→ 「保存」→ 「测试连接」
```

**不配 AI 也能跑**（系统降级为规则引擎），但准确率会明显下降。

## 3. 创建被检查单位

每次审计前，先建立被检查单位档案：

```
后台管理 → 单位 → 「创建单位」
```

| 字段 | 说明 |
|---|---|
| 单位名称 * | 如「XX市图书馆」 |
| 单位编码 | 内部编号，便于检索（选填） |
| 级别 | 「单位」或「部门」 |

## 4. 创建核查任务

```
左侧导航「核查任务」→ 右上「新建任务」
```

### 弹窗字段

| 字段 | 说明 |
|---|---|
| 被检查单位 * | 下拉选择已建单位（也可现场新建） |
| 任务名 * | 自由命名，如「2025年度XX单位内控评价复核」 |
| 评价年度 | 默认 2025 |
| 核查范围 | **全部指标**（推荐）/ 仅核查选定指标 |
| ⚡ 快速模式 | 勾选 = 跳过 AI 深度思考，全量约 5-10 分钟<br>不勾 = 精确模式，全量约 15-25 分钟 |

**建议**：首次使用 / 验证流程 → 勾选「快速模式」 + 选 5 个指标核查（2-3 分钟出结果）。

## 5. 上传佐证材料

### 5.1 进入任务详情页

任务列表点击「编号」或「任务名」→ 进入任务工作台 → 切换到「材料」标签

### 5.2 上传方式 A：单文件 + 绑定指标

```
绑定指标 ← 下拉选择该材料对应的指标（54 项之一）
材料文件 ← 选择文件
「上传」按钮 ← 点击
```

支持格式：`.pdf` `.docx` `.xlsx` `.txt` `.md`

### 5.3 上传方式 B（推荐）：文件夹批量上传

```
右上「批量上传文件夹」按钮 → 选择整个文件夹
→ 进度条显示每份文件解析情况
→ 上传完成后所有材料先归入"共享池"
```

### 5.4 AI 自动绑定指标

文件夹批量上传后，所有材料处于「未绑定」状态。点击：

```
「AI 自动绑定指标」按钮（材料表头）
```

系统会：
1. **关键词匹配**：文件名含「三重一大决策制度」等强关键词的直接绑定
2. **AI 阅读分类**：剩余未绑定的，LLM 读材料内容自动分类（30 秒-2 分钟）

**实测命中率 91-100%**。

完成后弹出 toast：「自动绑定 X 份（关键词 N + AI N），剩 Y 份请手动指定」。

### 5.5 手动绑定剩余材料

未绑定的材料行尾下拉框是**橙色高亮**，点开下拉手动选择对应指标。

下拉变更即自动保存。

### 5.6 全部绑定完才能触发核查

绿色横幅显示「**✓ 全部 23 份材料已绑定指标，可触发 AI 核查**」时，「触发 AI 核查」按钮变蓝可点。

若仍有未绑定，按钮为灰色 disabled 状态。

## 6. 触发 AI 核查

绑定完成后，点击「触发 AI 核查」按钮。

### 6.1 实时进度

任务工作台顶部出现进度卡片：

```
🔄 AI 核查中...  ⚡ 快速模式      17 / 54 项
[========================------]   31%
当前指标：I-17 预算预警整改
```

进度每 3 秒自动刷新（不需手动 refresh）。

### 6.2 跑完后自动跳转到「核查发现」标签

提示「AI 初核完成，共 95 条疑点：高 22 / 中 15 / 低 58」。

### 6.3 防呆

**已完成的任务再点「触发 AI 核查」会变成「重新核查」按钮**，点击前弹 confirm 对话框：「重新核查将清空已有疑点和工作底稿，确定继续吗？」

避免误触发覆盖已有数据。

## 7. 复核「核查发现」（Findings）

### 7.1 五维度统一展示

「概览」标签底部显示 5 个维度的疑点数：

| 维度 | 说明 |
|---|---|
| 真实性问题 | 公章 / 签字 / 年度 / 正式性 |
| 完整性问题 | 要素 / 章节 / 材料缺失 |
| 合规性问题 | 制度 / 流程 / 法规违规 |
| 重复性问题 | 跨单位材料重复 |
| 匹配性问题 | 材料与指标不匹配 |

### 7.2 单条复核

「核查发现」标签 → 左侧列表点疑点 → 右侧详情显示：

- 疑点描述
- 法律依据（自动 RAG 检索）
- 整改建议

底部有 3 个按钮：

| 按钮 | 含义 | 对评分的影响 |
|---|---|---|
| **确认** | 这条疑点成立 | 按 100% 扣分 |
| **忽略** | 这条疑点无效 | 不扣分 |
| **调整** | 部分成立 / 程度减轻 | 按 50% 扣分 |

### 7.3 ⭐ 批量忽略某类疑点

「核查发现」顶部有 5 个维度按钮：

```
共 95 条未复核：[忽略所有 真实性问题(22)] [忽略所有 完整性问题(57)] ...
```

**典型用法**：48 条低风险的「未上传材料」提示对你来说不算"真问题"，一键忽略，剩下的才是真信号。

## 8. 工作底稿

### 8.1 进入底稿

任务工作台 → 「工作底稿」标签 → 看到 54 行表格。

### 8.2 列结构

```
# | 分类 | 名称 | 核查要点 | 扣分规则 | 材料判定（5 对 chip）| 标准 | 核查前 | 核查后 | 调整得分说明
```

### 8.3 在线编辑

只要底稿不是「已定稿」状态，**任何单元格都可编辑**：

- **核查后得分**：数字输入框，blur 自动保存
- **调整得分说明**：textarea，blur 自动保存
- **材料判定 chip**：点击切换正向/负向（5 对中各选一边）

保存状态显示在汇总行右侧（「✓ 已保存」/「保存中…」/「✗ 失败」）。

### 8.4 状态机

| 状态 | 含义 | 谁能改 |
|---|---|---|
| **草稿（AI 生成）** | AI 跑完默认状态 | 审查员 |
| **复核中** | 审查员首次编辑后自动进入 | 审查员 |
| **已定稿** | 完成复核，锁定为只读 | 仅超级管理员可解锁 |

### 8.5 完成复核

确认底稿无误后：

```
工作底稿顶部「✓ 完成复核，定稿」按钮 → 弹 confirm → 确定
→ 状态变为「🔒 已定稿（只读）」
→ 所有单元格变为不可编辑
```

### 8.6 解锁（仅超级管理员）

误定稿了？切换到超级管理员账号：

```
工作底稿顶部「解锁底稿」按钮 → 弹 confirm
→ 状态退回「复核中」可继续编辑
```

## 10. 常见操作

### 10.1 删除任务

任务详情页右上「**删除任务**」红色按钮 → 弹 confirm → 确定。

⚠️ **会一并清除**：该任务下所有材料文件、AI 检出的 Findings、工作底稿。**不可恢复**。

### 10.2 修改任务材料绑定

「材料」标签 → 行尾下拉重新选择指标 → 自动保存。
之后**必须重新核查**（点「重新核查」按钮）才能让新绑定生效。

### 10.3 法规库内容预览

「法规库」→ 任意一条点「👁 查看」→ 在线阅读全文（不用下载）。

### 10.4 评价指标查看与下载

「评价指标」→ 任意指标点「👁 查看」→ 看核查要点、扣分规则全文。
右上「导出当前列表」→ 下载 JSON。

### 10.5 改自己的密码

页面**左下角**头像旁边有一把「钥匙」图标 → 点开弹窗 → 输新密码。

### 10.6 全部用户管理

「后台管理 → 用户」表格：

| 操作 | 图标 | 说明 |
|---|---|---|
| 改密码 | 🔑 | 超级管理员可改任意用户密码 |
| 启用 / 停用 | ⏸ | 临时禁用账号 |
| 删除 | 🗑 | 软删除（用户名前加 `deleted_` 前缀）|

## 12. 常见问题（FAQ）

**Q：AI 核查跑得很慢，怎么办？**
A：默认精确模式约 15-25 分钟。建议日常用「快速模式」（创建任务时勾选 ⚡），约 5-10 分钟。或者用「选定指标」核查关键 5-10 项。

**Q：上传材料后 AI 自动绑定一个也没命中怎么办？**
A：系统会优先 LLM 阅读分类（不是仅看文件名）。检查 **后台管理 → 大语言模型** 的 API Key 是否配置正确。没配 LLM 的话只会走关键词匹配，命中率会低。

**Q：「触发 AI 核查」按钮是灰色点不了？**
A：检查「材料」标签上方的横幅是不是绿色「✓ 全部 X 份材料已绑定指标」。如果还有橙色未绑定行，需要先用「AI 自动绑定」或手动下拉指定。

**Q：跑了一半想停掉怎么办？**
A：刷新页面看进度。如果真的需要停：联系管理员在服务器执行 `docker compose restart worker`（会强制中断当前任务）。

**Q：底稿定稿后发现要改，怎么办？**
A：找超级管理员账号，进底稿点「解锁底稿」按钮。

**Q：浏览器关掉再来还能看到原来的任务吗？**
A：能。所有数据存在服务器，登录回来就在。

**Q：被检查单位用户能看到其它单位的数据吗？**
A：不能。系统会按 `unit_id` 隔离，被检查单位用户只能看到自己单位的任务。

**Q：忘记密码？**
A：联系超级管理员在「后台管理 → 用户」里点钥匙图标重置。

**Q：浏览器显示「网络错误」？**
A：可能服务器掉线。管理员 ssh 上服务器跑 `docker compose ps` 看哪个服务挂了，再 `docker compose restart 服务名`。
